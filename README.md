# ☁️ IoT Weather Monitoring using ESP8266 and ThingSpeak

## 📌 Project Overview

This IoT-based Weather Monitoring System uses an ESP8266 NodeMCU and DHT11 sensor to collect real-time temperature and humidity data. The sensor readings are uploaded to the ThingSpeak cloud platform, where they are visualized as live charts for remote monitoring and analysis.

## 🚀 Features

- Real-Time Temperature Monitoring
- Real-Time Humidity Monitoring
- Cloud Data Logging with ThingSpeak
- Live Graph Visualization
- Wireless Data Transmission
- Remote Monitoring from Anywhere
- IoT-Based Environmental Monitoring

## 🛠️ Hardware Components

- ESP8266 NodeMCU
- DHT11 Temperature & Humidity Sensor
- Jumper Wires
- USB Cable
- Wi-Fi Network

## 🔌 Circuit Connections

| DHT11 Pin | ESP8266 Pin |
|-----------|------------|
| VCC | 3.3V |
| GND | GND |
| DATA | D4 (GPIO2) |

## 💻 Software Used

- Arduino IDE
- ESP8266 Board Package
- ThingSpeak Cloud Platform
- DHT Sensor Library
